## 📝 О проекте

**Цифровая доска** — это легковесный веб-инструмент для быстрого создания наглядных схем, зарисовок и пояснений. Идеально подходит для преподавателей, лекторов и всех, кому нужно визуально объяснять материал.

Просто откройте в браузере и начинайте рисовать — никакой установки, регистрации или сервера не требуется.

## ✨ Возможности версии 0.0.1

| Функция | Описание |
|---------|----------|
| 🖌️ **Маркер** | Основной инструмент для рисования от руки |
| 🎨 **Палитра цветов** | 6 предустановленных цветов + любой цвет через пикер |
| 📏 **Регулировка толщины** | От 2 до 40 пикселей, плавный ползунок |
| 🧽 **Очистка холста** | Одним кликом удалить всё нарисованное |
| 📱 **Поддержка тач-экранов** | Рисуйте пальцем на планшетах и смартфонах |
| 💾 **Сохранение состояния** | Рисунок не пропадает при изменении размера окна |

## 🖱️ Как пользоваться

1. **Выберите цвет** — кликните на цветной кружок или используйте пикер
2. **Настройте толщину** — двигайте ползунок «Размер»
3. **Нажмите и тяните** на белом поле, чтобы рисовать
4. **Очистить доску** — кнопка «Очистить всё»

> 💡 *Совет: используйте для объяснения алгоритмов, блок-схем, структур данных и любых других визуальных концепций.*
