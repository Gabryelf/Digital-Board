## 📝 О проекте

**Цифровая доска** — это легковесный веб-инструмент для быстрого создания наглядных схем, зарисовок и пояснений. Идеально подходит для преподавателей, лекторов и всех, кому нужно визуально объяснять материал.

Просто откройте в браузере и начинайте рисовать — никакой установки, регистрации или сервера не требуется.

## 🖱️ Как пользоваться

### Основы работы
1. **Выберите инструмент** — откройте выпадающее меню и выберите нужный
2. **Настройте цвет** — кликните на цветной кружок или используйте пикер
3. **Регулируйте толщину** — двигайте ползунок «Размер»
4. **Рисуйте** — нажмите и тяните на поле

### Работа с инструментами

| Инструмент | Действие |
|------------|----------|
| Маркер, Ластик | Нажмите и ведите — непрерывное рисование |
| Линия, Прямоугольник, Круг, Треугольник | Нажмите в одной точке, отпустите в другой |
| Заливка | Кликните внутри замкнутой области |

### Быстрые слоты

1. **Добавить инструмент в слот** — нажмите на слот с плюсиком (+)
2. **Использовать инструмент** — кликните по слоту левой кнопкой
3. **Удалить из слота** — кликните правой кнопкой по слоту
4. **Настройка количества слотов** — через меню настроек (иконка ⚙️)

### Множественные доски

- **Переключение** — используйте кнопки ◀ и ▶ в панели инструментов
- **Счётчик** — показывает текущую доску (например, 3 / 5)
- **Автосохранение** — каждая доска сохраняется автоматически

### Настройки

1. Нажмите иконку ⚙️ (шестерёнка)
2. Измените количество слотов (1-10)
3. Измените количество досок (1-20)
4. Нажмите «Применить» — изменения вступят в силу

### Тёмная тема

- Нажмите на иконку луны/солнца 🌙☀️ в правой части панели
- Цвет рисунка адаптируется при переключении

> 💡 *Совет: используйте для объяснения алгоритмов, блок-схем, структур данных и любых других визуальных концепций.*
